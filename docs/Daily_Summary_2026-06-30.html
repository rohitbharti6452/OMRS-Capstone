How to Run Tests Manually
# 1. Navigate to the Maven project
cd /path/to/OMRS-Capstone/openmrs-qa-automation

# 2. Run all tests
mvn clean test

# 3. API tests only (no browser)
mvn test -Dtest=PatientApiTest

# 4. UI tests — visible browser (watch it run)
mvn test -Dtest=LoginUITest -Dheadless=false

# 5. UI tests — headless (CI-style)
mvn test -Dtest=LoginUITest -Dheadless=true

# 6. BDD scenarios (once Component 5 is built)
mvn test -Dtest=RunCucumberTest
